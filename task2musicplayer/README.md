# Task 2 - Music Player (HTML + JavaScript)

## Features

- Search tracks by title/artist/category
- Categorize (category filter)
- Featured filter + Featured banner
- Play / Pause
- Skip (Prev / Next)
- Shuffle
- Repeat current track
- Volume control (and mute via keyboard `M`)

## How to run

Just open `index.html` in a browser.

> Note: Audio uses public sample MP3 URLs from soundhelix.com.
> If your browser blocks cross-origin audio or the links change, replace `src` URLs inside `index.html` under `const TRACKS = [...]`.
